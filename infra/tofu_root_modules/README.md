# tofu_root_modules

This directory contains root module directories. In the upstream bloom-housing/bloom repository, it
is intentionally empty so that root module changes to the Exygy owned deployments do not affect
downstream forks when they merge changes. The root modules for the Exygy owned deployments are
maintained in https://github.com/bloom-housing/bloom-exygy/tree/main/infra/tofu_root_modules.
