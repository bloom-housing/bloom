# Neighborhood Amenities - enableNeighborhoodAmenities

## Name

`enableNeighborhoodAmenities`

## Description

When true, the 'neighborhood amenities' section is displayed in listing creation/edit and the public listing view.

## Additional Information

- The amenities can be switched to dropdowns if the [enableNeighborhoodAmenitiesDropdown](./enableNeighborhoodAmenitiesDropdown.md) feature flag is turned on
