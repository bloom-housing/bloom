# Regions - enableRegions

### Name

`enableRegions`

### Description

When true, the region can be defined for the building address.

### Additional Information

- When enabled, the neighborhood turns from a free form box to a drop down that is associated to the selected region
- The Regions and Neighborhoods are currently hard coded to Detroit neighborhoods and regions
