# Swap Community Type with Programs - swapCommunityTypeWithPrograms

### Name

`swapCommunityTypeWithPrograms`

### Description

When true, the programs section on the frontend is displayed as community types.

### Additional Information

Functionally this is repurposing the programs to be displayed as "Community types" in the partner and public sites. This also means that the "Community types" (reserved_community_types) is hidden and can't be added to listings.
