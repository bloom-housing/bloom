# enableFilterByCounty

## Name

`enableFilterByCounty`

## Description

When true, the filter drawer on the public site includes the option to filter listings by county

## Additional Information

This is a dynamic filter populated by the primary jurisdiction's subJurisdictions. In order to appear on the list a jurisdiction must by a sub jurisdiction of the jurisdiction used for the public site.
