# enableWaitlistLottery

## Name

`enableWaitlistLottery`

## Description

When true, jurisdiction supports lotteries for waitlist opportunities

## Additional Information

This adds a new `ReviewOrderTypeEnum` value of `waitlistLottery` that can be used by both the API and frontends
