# What to Expect Additional Field - enableWhatToExpectAdditionalField

### Name

`enableWhatToExpectAdditionalField`

### Description

When true, the what to expect additional field is displayed in listing creation/edit form on the partner site.

### Additional Information

The additional information for the "What to Expect" section is saved as a separate field on the listing. This is so that it can be displayed independently from the rest of the section content.
