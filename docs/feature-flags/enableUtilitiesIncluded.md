# Utilities Included - enableUtilitiesIncluded

### Name

`enableUtilitiesIncluded`

### Description

When true, the 'utilities included' section is displayed in listing creation/edit and the public listing view.

### Additional Information

The options are hardcoded with the current options being:

- Water
- Trash
- Electricity
- Phone
- Gas
- Sewer
- Cable
- Internet
