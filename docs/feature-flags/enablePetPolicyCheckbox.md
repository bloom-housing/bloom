# enablePetPolicyCheckbox

## Name

`enablePetPolicyCheckbox`

## Description

When true, the pet policy field in the listing form is displayed as checkboxes instead of a text area

## Additional Information

The policy disclaimer shown on the public site (below the pets policy) is configurable and won't appear if no policy is present
